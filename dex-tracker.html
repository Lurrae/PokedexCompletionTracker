Lurrae's Living Dex Completion Tracker

This gigantic page contains dozens of 30-slot boxes just like those in Pokemon HOME!
With this, you can easily keep track of what Pokemon you've caught for something like a living dex. Click a Pokemon to mark it as "seen", then click it again to mark it as "caught"! Click a third time and it'll reset back to unseen.

Here's a key for what all the background and outline colors mean:

  • Dark red outline - These Pokemon cannot be stored in HOME in any meaningful capacity. Primarily, this includes forms that rely on held items (like Mega Evolutions) or fusion forms.
  • Yellow outline - These Pokemon can't be stored in HOME directly, but there is some factor that can differentiate them. For example, Gigantamax Pokemon have the Gigantamax Factor, and Meloetta-Pirouette has Relic Song.
  • Gray background - This is a Pokemon which, as of 11/05/2025, is only available in Pokemon GO. This excludes anything that could be obtained through events or is available in non-Switch games.
  • Red background - This is a Pokemon which, as of 11/05/2025, is completely unavailable in any Switch-era game, usually because it was a limited-time event that is no longer available (such as Zarude or Magearna)
  • Orange background - This is a Pokemon that either requires online features (such as Union Circles in S/V) or are only available in a game which you haven't marked as owning.
  • Yellow background - This is a Pokemon you've marked as "seen". I use this to track Pokemon I've acquired in a game but have not yet transfered to HOME.
  • Green background - This is a Pokemon you've marked as "caught", meaning it's been stored in HOME. Good job!
